### for loop:

#### continue; - Давталтыг тухайн алхамыг алгасаж дараагийн алхамд шилжүүлнэ. Жишээ: давталт дотор i -н утга 2 үед алгасна
```sh
for(let i=0; i <= 3; i++)
 if(i === 2)
 continue;
 console.log(i);
```

#### break; - Давталтыг шууд дуусгана. Жишээ: давталт дотор i -н утга 2 үед лавталт шууд дуусна.
```sh 
for(let i=0; i <= 3; i++)
 if(i === 2)
 break;
 console.log(i);
```
