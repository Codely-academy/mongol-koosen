### Жиших үйлдлүүд (Comparison Operators)

`x = 3`

- x > 3 `=> false`
- x >= 3 `=> true`
- x < 3 `=> false`
- x <= 3 `=> true`
- x === 3 `=> true` (Төрөл + утга), x == 3 (Зөвхөн утга)
- x !== 3 `=> false`
